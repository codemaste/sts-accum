Аккумуляторы для компьютерной техники

19.10.2010 | Автор: admin

Электрическая энергия сохраняемая аккумулятором является движущей силой для большинства современных электронных устройств, обеспечивающей стабильную работу на протяжении длительного времени без привязки к стационарной электрической сети. Именно аккумулятор дает современным ноутбукам, кпк, коммуникаторам и сотовым телефонам мобильность, делает их незаменимыми повседневными помошниками.
